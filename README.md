## 实现的功能

这是一个bookdown写作模板，已经实现了如下功能：

- [x] github action自动编译代码、部署到网页。
- [x] 兼容R包的结构。`Rmarkdown`可以调用该R包中的函数。
- [x] require2自动安装和加载R包（若未安装，则`pak::pkg_install`装之）。
- [ ] 参考文献
- [ ] 交叉引用
